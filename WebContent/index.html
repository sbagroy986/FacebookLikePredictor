Anant Jain - 20140xx
Shrey Bagroy - 2014099

Final project for the Advanced Programming course

This project aims at predicting the number of likes on a Facebook post.

It uses a users last 100 posts to estimate this value. This data is fetched using the Facebook Graph API. An OAuth access token is used, for which the user will need to log into their Facebook account and give this app the required permissions.
